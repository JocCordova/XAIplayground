<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. `git clone <repo>`
2. `cd <repo>`
3. `pip install virtualenv` 
4. `virtualenv venv` 
5. `venv/bin/activate` 
6. `pip install -r requirements.txt`
